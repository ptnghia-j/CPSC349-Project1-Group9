Project's Description

In this project, we built a website presenting a collection of photos capturing mother natures beauty. The project required Node.js and NPM. We also used Eleventy static site generator to quickly generate sites with templates. Nu Html Checks are used to verify the html codes.

The code written to be checked at local host is on main branch. The code published to Github Page is on deployPage branch.

Pexels

The photos we used were from Pexels. Pexels is a website offering freely-usable photographs shared by other creators and can be a useful soruce for website design.
